# Twitta

## Twitta介绍
一个仿 `Twitter` 后端项目

## Twitta项目功能点
- [x] 单点登陆
- [x] 人脸登录
- [x] 权限验证
- [x] 用户相关
- [x] 推文相关
- [x] 全局搜索
- [x] 交友私信
- [x] 关注粉丝
- [x] 朋友聊天
- [x] 点赞评论
- [x] 收藏推文
- [x] GRpc服务

# Quick Start
## Docker
```bash
docker-compose up -d
```
## Local
```bash
make run
```

# Swagger
## Generate
```bash
make swagger
```
## Url
```bash
http://localhost:6565/api/twitta/swagger/index.html
```
